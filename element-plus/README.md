## 📋 命名规范

所有文件都按照以下格式命名：
```
element-plus-{分类}-{组件名}.md
```

### 分类说明
- **`basic`**: 基础组件 - 按钮、边框、颜色、容器、图标等
- **`form`**: 表单组件 - 输入框、选择器、复选框、单选框等
- **`data`**: 数据展示组件 - 表格、卡片、标签、进度条等
- **`navigation`**: 导航组件 - 菜单、标签页、面包屑等
- **`feedback`**: 反馈组件 - 对话框、消息提示、通知等
- **`config`**: 配置组件 - 全局配置等
- **`other`**: 其他组件 - 分割线、水印等
